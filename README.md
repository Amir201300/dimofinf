<!-- Installation -->
### :gear: Installation

Clone the project

```bash
  git clone https://github.com/Amir201300/dimofinf
```

Go to the project directory

```bash
  cd dimofinf
```

Install dependencies

```bash
  composer install
```

Install migration and seeds

```bash
  php artisan project:install
```

Run test

```bash
  php artisan test
```


Start the server

```bash
  php artisan serve
```
